## ?? Швидкий запуск на GitHub Pages

### 1. Створіть репозиторій
- Зайдіть на [GitHub](https://github.com)
- Натисніть зелений кнопку **"New repository"**
- Назвіть його (наприклад, `interno-invicta`)
- Виберіть **Public**
- Натисніть **"Create repository"**

### 2. Завантажте файли
